# `false` - Command Reference

> Returns a `false` value

## Description

Returns a `false` value.

## Usage

    false -> <stdout>

## Examples

By default, `false` also outputs the term "false":

    » false
    false
    
However you can suppress that with the silent flag:

    » false -s

## Flags

* `-s`
    silent - don't output the term "false"
