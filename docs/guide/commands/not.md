# `!` (not) - Command Reference

> Reads the STDIN and exit number from previous process and not's it's condition

## Description

Reads the STDIN and exit number from previous process and not's it's condition.

## Usage

    `<stdin>` -> ! -> `<stdout>`

## Examples

    » echo "Hello, world!" -> !
    false

    » false -> !
    true

## Synonyms

- `!`
