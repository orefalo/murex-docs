    Murex.Rocks

    An intuitive, typed and content aware shell from the 2020s.

    Getting Started 💡Rosetta StoneDocs

    Easy to install

    • PKGBuild file https://aur.archlinux.org/cgit/aur.git/plain/PKGBUILD?h=murex

    • AUR Package https://aur.archlinux.org/packages/murex

    • Brew brew install murex

    • MacPorts sudo port install murex

    • More options are available in the INSTALL document.

    A Modern shell for the rest of us

    Murex carries tons of unique features for you.

    Content Aware

    natively manipulate JSON, TOML, YAML, XML, Markdown structures

    Objects and Maps

    use maps, hashed, lists and dictionaries

    Arrays manipulation

    native builtins allows the proper manipulation of arrays

    Scalar expression

    the shell parses variables as expression to get around accidental bugs due to spaces

    Public & Private functions

    private functions restrict scope visibility to nearest module or source file

    Type inference

    the shell tries to infer the data type of a variable or pipelines it manages

    Enhanced pipelines & redirection

    supports sending typed information to compatible functions via redirection and pipelines

    Type casting and formats

    change the meta-data about how an information should be read or displayed

    Inline spellchecking

    identifies typing spelling errors as underlined

    Smart Autocomplete

    use `TAB` to complete commands and parameters

    Hint text

    gives clues to the user without distractions

    Syntax highlighting

    in the interactive terminal, syntax is highlighted and can be pipelined

    Syntax Completion

    balances and auto-close brackets et accolades

    65 builtins commands

    full features builtins allow fast execution and portability
