    An intuitive, typed and content aware shell for the 2020s and beyond.

    A Modern shell for the rest of us

    Murex carries tons of unique features...

    Content Aware

    natively manipulate JSON, TOML, YAML, XML, Markdown structures

    Objects and Maps

    use maps, hashed, lists and dictionaries

    Arrays manipulation

    native builtins allows the proper manipulation of arrays

    Scalar expression

    the shell parses variables as expression to get around accidental bugs due to spaces

    Public & Private functions

    private functions restrict scope visibility to nearest module or source file

    Type inference

    the shell tries to infer the data type of variable and pipelines it manages

    Enhanced pipelines & redirection

    supports sending typed information to compatible functions via redirection and pipelines

    Type casting and formats

    change the meta-data about how an information should be read or displayed

    Inline spellchecking

    quickly identify typing spelling errors with underlines text

    Smart Autocomplete

    use `TAB` to auto-complete commands and parameters

    Hint text

    the hinttext gives clues to the user without any distractions

    Syntax highlighting

    in the interactive terminal, syntax is highlighted and style can be piped to the next built-in

    Syntax Completion

    balances and auto-closes brackets and accolades

    65 builtins commands

    full features builtins allow fast execution and portability
